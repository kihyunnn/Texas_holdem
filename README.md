# 🎲 Texas Hold'em Tracker

**AI-Powered Poker Game Recorder & Analyzer**

실시간 게임 기록, AI 코멘터리, 통계 분석, 플레이어 비교, 업적 시스템을 갖춘 종합 홀덤 추적 웹 애플리케이션입니다. Docker를 통해 어디서든 쉽게 배포하고 실행할 수 있습니다.

---

## ✨ 주요 기능

### 1. 게임 기록 & AI 분석
- ✅ **간편 기록**: 승자, 핸드(족보), 팟 금액만 입력하면 끝
- ✅ **AI 코멘터리**: GPT-4o-mini가 상황에 맞는 재치 있는 해설 제공
- ✅ **기록 관리**: 잘못 입력한 게임 삭제 가능

### 2. 스마트 필터링
- ✅ **날짜 필터**: 오늘 / 전체 / 기간 지정 조회
- ✅ **조건 검색**: 특정 플레이어, 특정 핸드(족보)로 필터링

### 3. 통계 & 시각화
- ✅ **오늘의 세션**: 매일 자정 자동 초기화되는 데일리 대시보드
- ✅ **실시간 차트**: Pot 추이 그래프, 승리 핸드 분포 (Chart.js)
- ✅ **리더보드**: 실시간 순위 및 수익 현황

### 4. 플레이어 분석 & 라이벌
- ✅ **심층 분석**: 승률, 총 획득액, 주력 핸드, AI 플레이 스타일 분석
- ✅ **라이벌 비교**: 두 플레이어 간 상대 전적 및 AI 대결 구도 해설
- ✅ **업적 시스템**: 14종의 뱃지 (첫 승리, 마스터, 로열 로더 등)

### 5. UI/UX
- ✅ **모던 디자인**: 직관적인 카드형 UI, 트렌디한 그라데이션
- ✅ **다크/라이트 모드**: 눈이 편안한 테마 지원
- ✅ **완전 반응형**: 모바일, 태블릿, 데스크톱 완벽 지원

## 🚀 Docker로 바로 실행하기 (추천)

Docker가 설치된 환경이라면 가장 쉽고 빠르게 실행할 수 있습니다.

### 1. 프로젝트 다운로드
```bash
git clone <repository-url>
cd Texas_holdem
```

### 2. 환경 변수 설정
OpenAI API 키 설정을 위해 `.env` 파일을 생성합니다.
```bash
cp .env.example .env
# .env 파일을 열어 OPENAI_API_KEY 값을 입력하세요.
# 예: OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxx
```

### 3. 실행
```bash
docker-compose up -d --build
```
- `-d`: 백그라운드 실행
- `--build`: 이미지를 새로 빌드 (최초 실행 시 권장)

### 4. 접속
브라우저에서 `http://localhost:5000` (또는 서버 IP:5000)으로 접속하세요.

## 🛠️ 로컬 개발 환경 실행

Python이 설치된 환경에서 직접 실행하는 방법입니다.

### 1. 가상환경 생성 및 패키지 설치
```bash
# 가상환경 생성
python -m venv venv

# 활성화 (Windows)
venv\Scripts\activate

# 활성화 (Mac/Linux)
source venv/bin/activate

# 패키지 설치
pip install -r requirements.txt
```

### 2. 실행
```bash
python app.py
```
서버가 `http://localhost:5000`에서 실행됩니다.

## 📚 주요 기능 가이드

### 탭 네비게이션
- **Games**: 전체 게임 히스토리 (필터 사용 가능)
- **Stats**: 오늘의 현황, 추이 그래프, 리더보드
- **Players**: 플레이어별 상세 통계, AI 인사이트, 업적
- **Rivalry**: 두 플레이어 간 전적 비교
- **Hands**: 포커 족보 확인 (이미지)

### 업적 시스템
플레이어는 다음과 같은 업적을 달성할 수 있습니다:

| 업적 | 조건 | 뱃지 |
|------|------|------|
| 첫 승리 | 첫 승리 기록 | 🎉 |
| 베테랑 | 10회 이상 승리 | 🎖️ |
| 마스터 | 50회 이상 승리 | 👑 |
| 부자 | 총 10만원 이상 획득 | 💰 |
| 백만장자 | 총 100만원 이상 획득 | 🤑 |
| 로열 로더 | 로열 플러시 승리 | 👑 |
| 스트레이트 플러시 | 스트레이트 플러시 승리 | 🌈 |
| 블러퍼 | 폴드 승 5회 이상 | 🎭 |

## 🔧 배포 및 관리

### 데이터 백업
모든 데이터는 `data/poker.db` 파일에 저장됩니다. `data` 디렉토리만 백업하면 데이터를 보존할 수 있습니다. Docker 실행 시에도 볼륨 마운트를 통해 데이터가 유지됩니다.

### 로그 확인 (Docker)
```bash
docker-compose logs -f
```

### 컨테이너 중지
```bash
docker-compose down
```
